холодильник наполнен, дневник пишется. Началась обкатка (фаза 1.5) — за первый

## Решения (через развилки с Никитой)

- **Интерфейс:** группа на троих (Никита + Богдан + бот); бот слушает обычную
  переписку. Личка — вспомогательно, база общая. Риск №1 проекта — выгорание
  Никиты ⇒ принцип «бот никогда не требует действий».
- **LLM сразу:** Claude Haiku (`claude-haiku-4-5`), structured outputs, один вызов
  на сообщение; ошибка LLM не теряет данные (raw_messages + kind=unparsed).
- **Точный учёт остатков** — но выживаемый: автосписание из свободного текста,
  apply-then-correct (кнопка «Отменить»), NULL = «есть, количество неизвестно».
- **Состав MVP:** ядро + планы Богдана (/today). Фото/дайджест/корзина — потом.
- **Провайдер:** прямой Anthropic API; OpenRouter отклонён (не дешевле: та же
  цена + ~5% комиссия; ломает structured outputs). Ключ создан, баланс $5.
- **Токен:** деплой на текущем (засвечен в переписке 02–05.07); замена + приватность
  репо — фаза 1.5–2.

## Сделано

1. Скаффолд + регистрация в хребте (PROJECTS.md, корневой CLAUDE.md, gitignore).
2. Код: aiogram 3 + SQLite stdlib + anthropic; ~700 строк, 6 модулей; 26 юнит-тестов
   (db+service, без внешних зависимостей). Пойман баг: SQLite COLLATE NOCASE не
   сворачивает кириллицу → нормализация имён на Python.
3. Живой прогон LLM-парсера на реальных сообщениях переписки до деплоя — 3/3.
4. Деплой: `/opt/eda`, venv, systemd `eda-bot.service`, `.env` chmod 600;
   соседние сервисы не задеты. Обновление: `git pull && systemctl restart eda-bot`.
5. Обкатка, день 1:
   - Калибровка №1: «холодильник» = весь запас еды дома («не в холодильнике,
     а рядом» не отменяет учёт); числа прописью.
   - Калибровка №2 (клубника-инцидент): порция никогда не обнуляет упаковку
     (subtract без qty → «есть»); словесные поправки («нет, X не кончилось») →
     set; кончившиеся позиции включены в LLM-контекст. Принцип: недосписать
     безопаснее, чем ложно обнулить.
   - Метод калибровки: скрин от Никиты → кейс в scratchpad-скрипт → живой прогон
     с прод-ключом (фикс + регрессия) → pull+restart. Цикл — минуты.
6. Наполнение холодильника вживую: 10 позиций одним сообщением (граммовки,
   морозилка, «вне холодильника»).
