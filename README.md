# Eda

Телеграм-бот — общая кулинарная память для двоих: ученик готовит и отчитывается,
наставник-повар видит холодильник и историю, даёт планы.

- Свободный текст в группе → LLM (Claude Haiku, structured outputs) → структура:
  дневник еды (сытость/вкус), учёт холодильника с автосписанием и undo, планы готовки.
- Стек: Python, aiogram 3, SQLite (stdlib), Anthropic API.
- Принцип: ноль трения — бот ничего не требует, ошибки исправляются одной кнопкой.

## Запуск

```bash
cp .env.example .env   # заполнить токен, ключ, id пользователей
python -m venv .venv && .venv/bin/pip install -r requirements.txt
.venv/bin/python -m bot.main
```

Бота добавить в группу администратором (иначе он не видит обычные сообщения).

## Тесты

```bash
python -m unittest discover -s tests -v
```

Слои БД и логики — stdlib-only, тесты не требуют ключей и сети.
